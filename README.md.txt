## ‚ú® Features

AutoNaksha aims to provide a professional, clutter-free architectural planning experience with the following core functionalities:

### 1. Dynamic Generation
* **Input-Driven:** Generates layouts based on Plot Length (ft), Plot Width (ft), and Number of Bedrooms (1, 2, or 3).
* **Feature Filtering:** Includes options for Car Parking, Bike Parking, and Staircase inclusion.
* **Multiple Designs:** Automatically generates a minimum of **20 distinct layout variations** to provide users with options.

### 2. Premium Rendering
* **Technology:** Uses **HTML5 Canvas** for dynamic, scalable 2D top-view floor plan rendering.
* **Clarity:** Rooms are clearly labeled with the **Room Name** and calculated **Room Size (ft √ó ft)**.
* **Scalable Output:** Designs maintain proportional accuracy based on user input.

### 3. Multi-Color Premium UI
* **Aesthetic:** Modern, architect-style UI with a premium look and feel.
* **Palette:** Utilizes a **Dark Gradient Background** (Charcoal/Navy) contrasted with **Light Cards** and accented with **Teal** and **Soft Orange** for a high-end visual experience.
* **Typography:** Clean, modern `Poppins` typeface for readability.

### 4. Free & Instant Download
* **One-Click Download:** Users can download the generated designs instantly as a PNG image directly from the Canvas.
* **Zero Barrier:** No login, no registration, and no payment required.
